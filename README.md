## How to Use

1. Replace `logo.png` with your actual company logo
2. Update portfolio project images by replacing the placeholder sections
3. Customize colors in `styles.css` by modifying the CSS variables in `:root`
4. Add your backend API endpoint in `script.js` for the contact form
5. Update content as needed - all text is easily editable

## Customization Guide

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00b4d8;
    --accent-color: #03dac6;
}
```

### Update Content
All content is in `index.html`. Simply search for the section and update the text.

### Add Images
- Replace portfolio placeholder with actual images
- Add company logo as `logo.png` in the same directory

### Contact Form Backend
In `script.js`, uncomment and configure the fetch API call to connect to your backend.
